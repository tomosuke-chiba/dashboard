# Product Overview

歯科クリニック向け採用メディア管理ダッシュボード。複数の求人媒体（GUPPY、ジョブメドレー、クオキャリア）からのデータを一元管理し、採用活動を可視化するツール。

## Core Capabilities

- **マルチメディア統合管理**: GUPPY、ジョブメドレー、クオキャリアの3つの採用媒体を単一ダッシュボードで管理
- **KPI可視化**: 閲覧数、応募数、スカウトメール開封率などの採用指標をリアルタイムで追跡
- **目標管理**: 職種別（歯科医師、歯科衛生士、歯科助手等）の採用目標と進捗管理
- **自動データ収集**: 各媒体からのスクレイピングによるデータ自動取得

## Target Use Cases

- 複数の歯科クリニックを管理する採用担当者が、各媒体のパフォーマンスを横断的に把握
- クリニック管理者が自院の採用状況と目標達成度を確認
- 採用KPI（閲覧率、応募率）の日別・月別トレンド分析

## Value Proposition

求人媒体ごとに分散した採用データを集約し、一目で採用活動の全体像を把握できる。手動でのデータ収集作業を自動化し、採用担当者の業務効率を向上。
