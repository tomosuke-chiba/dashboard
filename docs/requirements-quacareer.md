# クオキャリア 要件定義書

## 1. 概要

| 項目 | 内容 |
|------|------|
| 媒体名 | クオキャリア（Quacareer） |
| URL | https://www.quacareer.com（要確認） |
| 特徴 | **プロフィール変更不可**（唯一の制約がある媒体） |
| 認証方式 | ID + パスワード（要確認） |

---

## 2. 採用の流れとKPI

### 2.1 採用ルート

**スカウトメール経由のみ**
```
スカウトメール送信
  ↓
開封（既読）  ← 取得可能
  ↓
返信
  ↓
採用
```

### 2.2 KPI指標

| 指標 | 重要度 | 備考 |
|------|--------|------|
| **スカウトメール開封率** | 最重要 | プラットフォームから取得可能 |
| **スカウトメール返信率** | 最重要 | どの文面が効果的か最適化 |

### 2.3 他媒体との違い

| 項目 | GUPPY | ジョブメドレー | クオキャリア |
|------|-------|----------------|--------------|
| プロフィール変更 | 可能 | 可能 | **不可** |
| 閲覧データ | あり | あり | 限定的 |
| スカウト開封率 | 取得困難 | 要調査 | **取得可能** |
| 最重要施策 | プロフィール最適化 | 検索順位UP | **スカウト文面最適化** |

---

## 3. 戦略的位置付け

### 3.1 クオキャリアの役割

プロフィール変更ができないため：

1. **スカウトメールの最適化に注力**
   - どの文面が開封されやすいか
   - どの文面が返信されやすいか
   - A/Bテストの実施

2. **他媒体との比較検証**
   - 同じ文面を複数媒体で送信
   - クオキャリアの開封率データを活用して効果検証

---

## 4. データ取得仕様

### 4.1 スカウトメールデータ

| 項目 | 詳細 |
|------|------|
| 送信数 | 管理画面から取得 |
| 開封数 | 管理画面から取得（**クオキャリア独自の強み**） |
| 返信数 | 管理画面から取得 |
| 更新頻度 | 1日4回 |

### 4.2 計算指標

| 指標 | 計算式 |
|------|--------|
| 開封率 | 開封数 / 送信数 |
| 返信率 | 返信数 / 送信数 |
| 開封→返信転換率 | 返信数 / 開封数 |

## 8. 活用方法

### 8.1 スカウトメール最適化サイクル

```
1. 現在の開封率・返信率を計測
2. 新しい文面でテスト送信
3. 結果を比較
4. 効果の高い文面を採用
5. 繰り返し
```

### 8.2 他媒体への知見展開

クオキャリアで効果の高かった文面を：
- GUPPYのスカウトメールに適用
- ジョブメドレーのスカウトメールに適用

---

## 9. 注意事項

### 9.1 プロフィール変更不可の影響

- 閲覧経由の採用ルートは最適化できない
- スカウトメールのみが施策の焦点
- 他媒体と比較して施策の幅が限定的

### 9.2 データ活用の優位性

- 開封率データが取得可能なのはクオキャリアの強み
- このデータを活用してスカウト文面のA/Bテストが可能
- 得られた知見を他媒体に展開できる
